TODO App (React + JSON storage)

This project contains a React frontend and a minimal Express backend that persists todos to a JSON file.

Quick start:

1. Install server deps and start server:

```bash
cd server
npm install
npm start
```

2. Install client deps and start frontend:

```bash
cd client
npm install
npm run dev
```

Server runs on port 4000 and frontend on port 5173 (Vite default).
